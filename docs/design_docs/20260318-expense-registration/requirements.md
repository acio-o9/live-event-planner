# Requirements: Expense Registration

**Status**: Draft
**Created**: 2026-03-18
**Developer**: maru

## What（何を作るか）

ライブイベントにかかった費用を登録・管理する機能。
参加メンバーが立て替えた金額を入力し、イベント全体の合計費用と一人当たりの負担額を自動計算する。

## Why（なぜ必要か）

集計作業と個人が立て替えている金額から一人当たりの負担額を参加者で簡単に出せるようにするため。
現状は手動で集計しており、ミスや手間が発生している。

## Requirements（要件）

### Functional Requirements（機能要件）

- [ ] 費用の登録（金額・立替者・カテゴリ・メモ）
- [ ] 費用の編集・削除
- [ ] 参加者ごとの立替金額の一覧表示
- [ ] イベント全体の合計費用の自動集計
- [ ] 参加人数で割った一人当たりの負担額の自動計算

### Non-Functional Requirements（非機能要件）

- [ ] 初期表示: 2秒以内
- [ ] テストカバレッジ: 80%以上
- [ ] レスポンシブ対応（スマートフォンから入力できる）

## Success Criteria（成功基準）

- [ ] 費用の登録・編集・削除が正しく動作する
- [ ] 合計金額・一人当たり負担額が正確に計算される
- [ ] 複数の立替者がいる場合も正しく集計される
- [ ] すべてのテストが通過する
- [ ] コードレビューで承認される
