# Requirements: Milestone CRUD

**Status**: Draft
**Created**: 2026-04-30
**Developer**: maru

## What（何を作るか）

ライブイベント詳細画面のマイルストーンタブにおいて、マイルストーンの作成・編集・削除を行えるUIと対応APIを実装する。
現状は閲覧とステータス変更のみ対応しており、新規追加・内容変更・削除ができない。

## Why（なぜ必要か）

イベント作成後にマイルストーンのタイトルや期日を変更できず不便。
イベント計画の変更に柔軟に対応するため、フルCRUD操作が必要。

## Requirements（要件）

### Functional Requirements（機能要件）

- [ ] マイルストーンの新規追加（タイトル・期日）
- [ ] 既存マイルストーンの編集（タイトル・期日）
- [ ] マイルストーンの削除（確認ダイアログあり）
- [ ] 一覧表示は期限が近い順（dueDate昇順、未設定は末尾）

### Non-Functional Requirements（非機能要件）

- [ ] 操作後の画面更新はサーバー再取得なくローカルstate更新で即時反映
- [ ] モーダルのフォームバリデーション（タイトル必須）
- [ ] レスポンシブ対応（既存ページに準拠）

## Success Criteria（成功基準）

- [ ] マイルストーンの追加・編集・削除がUIから正常に動作する
- [ ] 一覧が期限昇順（未設定は末尾）で表示される
- [ ] 削除時に確認ダイアログが表示される
- [ ] 既存のステータス変更・タスク管理機能が引き続き動作する
- [ ] 既存のライブイベントメンバー権限制御に準拠している
