# Requirements: Profile Edit

**Status**: Draft
**Created**: 2026-04-02
**Developer**: maru

## What（何を作るか）

ログイン中のユーザーが自分のニックネームと担当パートを変更できるプロフィール編集ページを作成する。
あわせて、メンバー一覧ページでは担当パートによるクライアントサイドフィルタリングができる。

## Why（なぜ必要か）

- 現在、ユーザーが自分の名前（ニックネーム）を変更できない
- 各メンバーがどの楽器（パート）を担当しているかが不明確

## Requirements（要件）

### Functional Requirements（機能要件）

- [ ] ログイン中のユーザーが自分のニックネームを編集・保存できる
- [ ] ログイン中のユーザーが自分の担当パートを選択・保存できる
- [ ] ニックネームは全角10文字（半角20文字相当）以内のバリデーションを実施
- [ ] ニックネームは空文字不可・前後空白トリム・特殊文字制限などの一般的なバリデーションを実施
- [ ] メンバー一覧ページで担当パートによるクライアントサイドフィルタリングが可能
- [ ] ページネーションは不要（全件表示）

### Non-Functional Requirements（非機能要件）

- [ ] 初期表示: 2秒以内
- [ ] レスポンシブ対応（モバイル〜デスクトップ）
- [ ] フォームバリデーションはリアルタイム（入力中）で表示

## Success Criteria（成功基準）

- [ ] ニックネームと担当パートが正常に更新・保存できる
- [ ] バリデーションエラーが適切に表示される（空文字・文字数超過など）
- [ ] 担当パートフィルターが正常に動作する（全件表示・パート別絞り込み）
- [ ] すべてのテストが通過する
- [ ] コードレビューで承認される
