# Requirements: Member Management

**Status**: Draft
**Created**: 2026-04-24
**Developer**: maru

## What（何を作るか）

メンバー一覧ページからメンバーの追加・ニックネーム編集・削除ができる管理機能を実装する。
追加時は `dummy-{uuid}` 形式の sub を自動生成し、ダミーメンバーとして登録する。
削除は論理削除（`deletedAt` フィールドを追加）で行い、削除済みメンバーは一覧に表示されない。

## Why（なぜ必要か）

間違って登録したメンバーを修正・削除したい。
管理者がメンバーを管理できる機能が不足しているため。

## Requirements（要件）

### Functional Requirements（機能要件）

- [ ] メンバー一覧からニックネームを編集できる
- [ ] メンバーを削除できる（確認ダイアログあり、論理削除）
- [ ] メンバーを追加できる（sub は `dummy-{uuid}` 形式で自動生成）
- [ ] 削除済みメンバーは一覧に表示されない

### Non-Functional Requirements（非機能要件）

- [ ] 操作後にメンバー一覧を即時リフレッシュ
- [ ] モバイル対応（レスポンシブ）
- [ ] API レスポンス: 1秒以内

## Success Criteria（成功基準）

- [ ] メンバーの追加・編集・削除が正常に動作する
- [ ] 削除時に確認ダイアログが表示される
- [ ] 削除済みメンバーは一覧に表示されない（論理削除）
- [ ] ログイン済みユーザーであれば全員操作可能
- [ ] すべてのテストが通過する
