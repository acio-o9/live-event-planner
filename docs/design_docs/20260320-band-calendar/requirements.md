# Requirements: Band Calendar

**Status**: Draft
**Created**: 2026-03-20
**Developer**: maru

## What（何を作るか）

バンドの集まる予定を共有するカレンダー機能。
ログインユーザー全員が全バンドの予定を閲覧できる。
予定には場所・時間・バンドを持ち、カレンダー上にはバンド名と場所が表示される。
ナビゲーションに「カレンダー」タブを追加し、直近2ヶ月の予定を確認できる。

## Why（なぜ必要か）

バンドごとの活動になりがちなため、音楽部所属メンバーがいつ集まるかわかることで横のつながりをより持てるようにしたい。

## Requirements（要件）

### Functional Requirements（機能要件）

- [ ] 月表示カレンダーで全バンドの予定を閲覧できる
- [ ] 直近2ヶ月（当月・翌月）の予定が表示される
- [ ] 予定にバンド・場所・開始日時・終了日時を登録できる
- [ ] ログインユーザー全員が閲覧可能
- [ ] 所属バンドの予定のみ登録・削除できる（他バンドは閲覧のみ）
- [ ] カレンダー上に「バンド名・場所」を表示（スペースに収まる範囲）

### Non-Functional Requirements（非機能要件）

- [ ] 初期表示: 2秒以内
- [ ] レスポンシブ対応（スマートフォンでも予定が確認できる）
- [ ] テストカバレッジ: 80%以上（計算・バリデーションロジック）

## Success Criteria（成功基準）

- [ ] バンドごとに予定を登録できる
- [ ] 所属バンドでしか予定を登録・削除できない
- [ ] 全ログインユーザーがカレンダーを閲覧できる
- [ ] すべてのテストが通過する
- [ ] コードレビューで承認される
