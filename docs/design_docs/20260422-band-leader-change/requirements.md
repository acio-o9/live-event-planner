# Requirements: Band Leader Change

**Status**: Draft
**Created**: 2026-04-22
**Developer**: maru

## What（何を作るか）

バンドのメンバー一覧からバンドマスター（リーダー）を選択・変更できるシンプルな機能を実装する。
あわせて、既存の「リーダー」という表示ラベルを「バンドマスター」に変更する。

## Why（なぜ必要か）

現在の仕様ではバンドを作成したユーザーが自動的にリーダーになるため、
運用上でバンドマスターを変更したいケースに対応できない。

## Requirements（要件）

### Functional Requirements（機能要件）

- [ ] バンドメンバー一覧モーダルで、非バンドマスターメンバーに「バンドマスターに変更」ボタンを表示する
- [ ] ボタン押下でそのメンバーをバンドマスターに変更し、旧バンドマスターは一般メンバーに降格する
- [ ] 「リーダー」という表示ラベルをすべて「バンドマスター」に変更する
- [ ] バンドマスターは常に1人（ロール変更はトランザクションで原子的に処理）
- [ ] バンドマスターの削除は引き続き禁止

### Non-Functional Requirements（非機能要件）

- [ ] API応答時間: 500ms以内（p95）
- [ ] バンドマスター変更はDB上でトランザクション処理し、整合性を保証する
- [ ] レスポンシブ対応（モーダル内UI、既存スタイルに準拠）

## Success Criteria（成功基準）

- [ ] バンドメンバー一覧モーダルから別メンバーをバンドマスターに変更できる
- [ ] 変更後、旧バンドマスターが一般メンバーとして表示される
- [ ] 「リーダー」表示がすべて「バンドマスター」に変わっている
- [ ] バンドマスターは削除できないままである
- [ ] すべてのテストが通過する
- [ ] コードレビューで承認される
